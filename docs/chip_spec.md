# アンミカ三麻 チップ [祝儀] 仕様書

実装は `src/lib/game3.ts` の `applyChipsOnHule` / `applyChipOall` / その他 局中変動箇所。

## 用語

- **チップ [chip]**: 麻雀点数とは別の祝儀通貨。 局単位で増減し、 半荘終了まで累積 [`chipLedger`]。
- **N オール**: ターゲットに他家 2 人から N ずつ徴収、 ターゲット +2N、 他家 -N each。 `applyChipOall(target, n)`
- **ロン放銃のみ**: 放銃者から winner に N、 他家は 0。

## 局中変動 [チップ ledger に即時反映]

### 春 [華牌 f1]

| 発生 | 効果 |
|---|---|
| 春 1 枚抜き [配牌 / ツモ] | +1 オール、 `haruActive` true |
| `haruActive` 中に他華 [f2/f3/f4] 抜き | +1 オール 追加発動 |
| 春 2 枚目抜き | 発動せず [アガリ時 春春扱い] |

### シュバリーチ宣言

- 当局のみ全祝儀 ×2 [`applyChipOall` 内で `n*2`]
- 半荘 1 回限定 [`shuvariUsed`]
- 非適用: サイコロ出目当て / 100-300 オール / トントンブー / 8万点+ / 冬冬金北 / 春 [2026-05-23 audit、 春はシュバ非適用]

## アガリ時集計 [`applyChipsOnHule`]

### 牌種ボーナス

| 種類 | 効果 |
|---|---|
| 赤 5 [m0/p0/s0、 金分除く] | +2 / 枚 オール |
| 金 5 [goldHand 合算] | +4 / 枚 オール |
| 抜きドラ [北抜き枚数] | +1 / 枚 オール、 4 枚で +4 オール [打点側 8 翻ボーナス済] |

### 華牌アガリ時集計

| 条件 | 効果 |
|---|---|
| 春 [f1 1 枚、 春春なし] | アガリ時、 抜いている華牌総数 × 1 枚オール |
| 春春 [haru >= 2] | アガリ時、 抜いている華牌総数 × 2 枚オール |
| 冬 [f4 1 枚] | +2 / 枚 オール [簡略アリス] |
| 冬冬 [f4 2 枚] | チューリップ [現物 + 上下牌ヒット、 実装済] |

### 役満祝儀 [ルール 7-1]

| 倍率 | ツモ | ロン |
|---|---|---|
| 役満 (damanguan=1) | 5 オール | 10 [放銃のみ] |
| ダブル (=2) | 5 | 10 |
| 五倍 (=3) | 7 | 12 |
| 六倍 (=4+) | 9 | 14 |

### 三倍満

- ロン / ツモ問わず +3 オール

### 面前役 [3-4]

| 役 | チップ | 払い方 |
|---|---|---|
| 面前ホンイツ | +5 | ツモ全員 / ロン放銃のみ |
| 面前チンイツ | +10 | 同上 |
| 二盃口 | +15 | 同上 |

### フィーバー立直

| tier | 効果 chip |
|---|---|
| single | +1 オール |
| double [7s+7p] | +2 オール |
| triple [7s+7p+7m] | +4 オール |

加えて打点 ±tier段アップ [LEVEL_TO_FANSHU 配列]。

### トントンブー

- 東 1 局 + 親アガリ + 他家トビ → +6 オール

### ぽっち効果

| 色 | 効果 |
|---|---|
| 緑 [正] | 通常受取 |
| 青 [正・祝儀倍] | アガリ時 chipLedger /2 倍加算 [簡略] |
| 黄 [逆] | 点数 + 祝儀 全逆払い [pochiPaymentMode 経由] |
| 赤 [逆・祝儀倍] | チップ逆払い [簡略] |

ツモ時の付与:
- 逆ぽっち [赤 / 黄] ツモ → `pochiPaymentMode` true (= payment mode)
- 正ぽっち [青 / 緑] ツモ → false [復帰]
- 白待ち [`tingpai` に z5] ならば payment mode 切替なし [回避]

## 半荘終了時 [`getFinalScore`]

- 終了時トップ n 万点 [defen >= 80000] → top に (2n - 10) chip オール、 例 8万=6 / 9万=8 / 10万=10
- ウマ:
  - トップ 40000 以上: +30 / 0 / -30
  - 全員 40000 未達: +45 / -15 / -30 [ただし返り東で継続するので通常到達しない]
- 最終スコア = (defen - 35000) / 1000 + ウマ + 累積 chip

## 実装ファイル

- `src/lib/game3.ts` — Game3 class
  - `chipLedger: Record<PlayerId, number>` — 累積、 半荘 reset せず
  - `haruActive: Record<PlayerId, boolean>` — 春 active flag
  - `applyChipOall(target, n)` — 「N オール」 [シュバリ active で ×2]
  - `applyChipsOnHule(result, winner, loser)` — アガリ時集計
  - `getFinalScore()` — 半荘終了時のウマ + 累積 chip
- `src/App.svelte` — UI、 各 player の chip を `💴 N` で表示
