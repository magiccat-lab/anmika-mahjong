## 0. 用語集

| 用語 | 意味 |
|---|---|
| アリス | 冬を抜いた状態で、アガリ時の現物牌（同じ種類・数字）をめくって当たれば祝儀。1枚ごとに2枚加算（副露時は1枚）。 |
| チューリップ | 冬2枚抜いた場合の強化版アリス。現物に加えて上下牌（±1）もヒット対象。ツモはオール、ロンは放銃者のみ。 |
| フィーバー立直 | 7p or 7s（または7m）を確定暗刻で立直する特殊役。立直者以外が強制ツモ切り、待ち切れるまで何度でもアガれる。 |
| ダブルフィーバー | 7s と 7p の両方が確定暗刻でリーチ。打点・祝儀ともに2倍。 |
| トリプルフィーバー | 7s・7p・7m の全てが確定暗刻でリーチ。打点・祝儀ともに4倍。 |
| シュバリーチ | 東風1回につき1度宣言可。成立時その局の祝儀が倍になる（一部除く）。 |
| オールスター | 赤5p・赤5s・金5p・金5s の4枚を全て揃えてアガリ。サイコロチャンス発生。 |
| トントンブー | 東一局で親だけがアガリ、そのままトビ終了。6枚オール祝儀。 |
| 神ポッチ | 正ぽっちがドラ表に出た状態。任意の牌を正ぽっちとして扱える。 |
| 金北 | 金色の北。アガリ時に、抜いている華（春夏秋冬）のいずれか1枚の効果を強化できる。 |
| サイコロチャンス（出目当て） | 特定条件で発生する祝儀ボーナス。宣言した出目をサイコロ2つを4回振って当てる。 |
| シュバサイコロ | シュバ権が使えないサイコロチャンス。流し役満・天地人和 等。 |
| 間八萬（かんぱーまん） | 7-9萬で8萬を間チャン待ちにして立直する特殊アガリ形。ツモ本役満＋サイコロチャンス、出アガリ8ハン。 |

## 1. 基本ルール

### 東風完全順位戦
- 35000点持ち40000点返し
- 2着クビあり（**2着**が40000点以上到達）: +30 / 0 / -30
- 2着クビなし（2着が40000点未達）: +45 / -15 / -30
- オーラス終了時誰も40000点未達なら返り東
- ノーテン親流れナシ、アガリ止め可

### 成立条件
- 後付けアリ、喰いタンあり
- 面前時以外、倒牌時に1ハンあればアガリ可能
- 面前時のダマアガリ不可（副露後のダマは可、国士・天和・地和・人和はダマOK）
- 数え役満は13ハン以上
- 符計算あり、ツモ損あり、ただし1000点加符
- ドラ表2枚、王牌20枚

### リーチ
- 0点持ちリーチ可（オープンリーチも可）。流局や他家アガリでトビ扱い
- リーチ後はカン材ツモで強制カン（待ちが変わらないカン可）

### アガリ・ロン・ツモ
- ダブロンあり、親アガリなら席順関係なく親続行
- 供託は上家取り
- 一本場2000点（ロン2000点加算、ツモ1000点オール加算）

### 牌構成
- 数牌の萬子は 1萬を「7萬」として扱う
- 7萬はヤオチュー牌とタンヤオ牌の両方の性質を持つ
- 国士無双・清老頭は7萬でもアガリ可
- 7萬確定暗刻でフィーバー立直可（後述）

## 2. 特殊牌

### 2-1. 赤・金
| 牌 | 効果 |
|---|---|
| 赤5p / 赤5s | ドラ、かつ 2枚加算 |
| 金5p / 金5s | ドラ、かつ 4枚加算 |

- オールスター: 赤2枚＋金2枚を全て揃えてアガリ → サイコロチャンス発生

### 2-2. 華牌（春夏秋冬）

共通:
- ドラではない、抜いても一発消えない、河に切れない（空気扱い）
- ドラ表示牌の華牌もアガリ時に抜いたものとして計算

#### 華牌 1枚の効果

| 華 | 効果 |
|---|---|
| 春 | 1枚オール。他の華牌を抜くたびに更に1枚オール加算。ただし春の2枚目は効果発動せず、アガリ時に発動（「春春」扱い） |
| 夏 | 打点ワンランクアップ（1ハン→2ハン→…→満貫→跳満→倍満→三倍満→役満→五倍満→六倍満） |
| 秋 | ドラ表をもう1枚めくれる。リーチ中なら裏ドラも見れる。秋と冬の両方ある場合は秋を先に発動。下段しか山が残っていない場合は使用不可 |
| 冬 | アリス（現物のみ）。1牌ヒットごとに2枚加算（副露時は1枚）。華牌は種類を問わず現物扱い |

#### 華牌 2枚（ダブル）の効果

| ダブル | 効果 |
|---|---|
| 春春 | 1枚オール ＋ 他の華牌1枚ごとに+1枚。例: 春春夏秋の4枚抜きで4枚オール |
| 夏夏 | 打点ツーランクアップ |
| 秋秋 | ドラ表＋2枚追加（リーチ時は裏ドラも2枚分）。金北強化時はハン数分祝儀追加 |
| 冬冬 | チューリップ（現物＋上下牌がヒット対象）。ツモはオール、ロンは放銃者のみ |

### 2-3. ぽっち（4色）

- リーチ後のみオールマイティー牌（強制高め取り、赤金不可）
- 「正ぽっち」「逆ぽっち」は色のグループ名（実体は下記 4 色）
  - 正ぽっち = 緑（代表）/ 青（祝儀倍の正ぽっちと呼ぶことあり）
  - 逆ぽっち = 黄（代表）/ 赤（祝儀倍の逆ぽっちと呼ぶことあり）

| 色 | 効果 |
|---|---|
| 青ぽっち | 祝儀倍（チップのみ倍、点数は倍にならない） |
| 赤ぽっち | 逆祝儀倍（チップのみ逆払い、点数は倍にならない） |
| 緑ぽっち | 通常受け取り |
| 黄ぽっち | 逆ぽっち（点数と祝儀がそのまま逆払い） |

補足:
- 白待ちの場合は通常白に取れる（逆ぽっち回避可）
- 白暗刻でツモらずカン選択可
- 白暗カンしてアガリ → サイコロチャンス（正ぽっちならツモかカン選択可）
- ぽっちをツモって祝儀0枚 → サイコロチャンス
- 正ぽっちがドラ表に出たら任意の牌にとれる（神ポッチ）
- 逆ぽっちツモ後トビは自トビ（トビ賞なし）
- フィーバー中の逆ぽツモ後放銃トビは自トビにならない（トビ賞あり）
- ぽっちこぼし（白ぽっちをこぼす）→ 即チョンボ＋効果処理あり
- ぽっちこぼし時に誰も立直かかっていない場合は正逆どちらも全員に対して効果消滅

### 2-4. 北

- 抜きドラ 1ハンかつ1枚、4枚抜きで8ハン（チップは4枚）
- 抜いても一発消えず、河には切れない
- 字一色・四喜和・国士無双のみ手牌使用可
- 抜いた際のポン不可、上記役満のみロン可

### 2-5. 金北

アガリ時に、抜いている華の効果を1つ強化できる。

| 強化先 | 効果 |
|---|---|
| 春 | もう一度春を発動 |
| 夏 | 夏夏にのみ適用可。2ランクアップが無くなる代わりに打点4倍 |
| 秋 | もう一度秋を発動。秋秋に適用するとハン数分祝儀追加（夏あり時は1枚につき2ハン扱い） |
| 冬 | アリス・チューリップともに上下段となる |

- 逆ぽっちツモ時は同卓者が強化先を選ぶ（抜け番不利の原則）
- フィーバー中は強化先選択は1回のみ（保留も可）

## 4. サイコロチャンス（出目当て）

### 4-1. 発生条件（一覧）

| 発生源 | 備考 |
|---|---|
| 本役満アガリ | ダブル・トリプルは2回・3回 |
| 白ぽっちを即ツモ | — |
| オールスターアガリ | 赤金4枚揃い |
| 四華・四北を抜いた上でのアガリ | — |
| 三連刻・三色同刻（面前時） | — |
| 白暗カンからのアガリ | — |
| ぽっちをツモって祝儀0枚 | — |
| 間八萬のツモ | — |
| 789三色を含む間八萬の出アガリ | ベース140枚 |
| 天和 / 地和 / 人和 | シュバサイコロ扱い |
| 流し役満 | シュバサイコロ扱い |

### 4-2. 四華・四北・八華まわりの特殊祝儀

| パターン | 祝儀 |
|---|---|
| 四華四北 または 八華 | 100枚オール ＋ 出目当て2回 |
| 八華四北 | 300枚オール ＋ 出目当て3回 |
| 鳴き四華 | 出目当て成功で35枚オール |
| 鳴き八華 | 出目当て成功で70枚オール |

### 4-3. 出目当ての方法

- ゾロ目以外の出目を宣言し、サイコロ2つを4回振る
- ゾロ目はリプレイ（回数消費なし）
- 通常ヒット: 1ヒットごとに70枚オール
- 天和・地和・人和・カラス・八連荘: 140枚オール
- 鳴き四華: 35枚オール
- シュバリー時のゾロ目連続ボーナス: 22→22枚、33→33枚、…、11→111枚オール

### 4-4. シュバサイコロ（シュバ権が使えないサイコロチャンス）
- 流し役満
- 天和・地和・人和

### 4-5. フィーバー中のサイコロチャンス
- 原則複数回アガリしても出目当ては複数回発生しない
- ただし本役満（四暗刻・カラス含む全本役満）のみアガリ回数分発生
- 四華アガリ後に四華四北に進化した場合は合算

## 5. フィーバー立直

### 5-1. 条件
- 手牌で 7p / 7s / 7m のいずれかが確定暗刻で、面前かつ立直可能
- 供託1000点

### 5-2. 手順
1. フィーバー立直宣言
2. 宣言牌でロンが発生しなければ手牌をオープン
3. 他家は待ち牌を晒す
4. 待ち牌が全て無くなっていなければ成立、全て無くなっていれば1人テンパイ流局

### 5-3. 効果
- 立直者以外は強制ツモ切り（聴牌していても）
- 待ちが山から無くなるまで何度でもアガリ可能
- 白ぽっちのみ残りの場合は待ちが山にあるとみなさない

### 5-4. ダブル・トリプルフィーバー
| 種類 | 条件 | 効果 |
|---|---|---|
| ダブルフィーバー | 7s と 7p の両方を完全暗刻でリーチ | 打点・祝儀ともに2倍 |
| トリプルフィーバー | 7s・7p・7m 全てを確定暗刻でリーチ | 打点・祝儀ともに4倍 |

### 5-5. フィーバー中の特殊処理

| 事象 | 処理 |
|---|---|
| 冬を抜いている | 任意のアガリ時にフィーバーを終了してアリス・チューリップ発動可 |
| 正ぽ → 逆ぽをツモ | 以降のアガリは全て逆払い（救済: 逆ぽ後に正ぽをツモると2倍を乗算して正に変換） |
| 王牌死にで流局 | 流し役満は不成立（フィーバー立直成立中は他家のアガリ・聴牌は一切成立しないため） |
| サイコロチャンス | 原則1回のみ。本役満（カラス・四暗刻含む）のみアガリ回数分発生 |

## 6. シュバリーチ

### 6-1. 基本
- 東風1回につき1度だけ、立直時に宣言可
- その局終了時まで受け取る全ての祝儀が倍（一部除く）
- 次の人の打牌までは宣言可能
- アガれなくても権利消費

### 6-2. シュバリー非適用の祝儀
- サイコロチャンス出目当て
- 四華四北・八華の100枚、八華四北の300枚
- トントンブー・8万点以上祝儀
- 冬冬金北

### 6-3. シュバ後の春
- シュバリー宣言後に抜いた春は効果が倍（既に抜いてある分は倍にならない）

## 8. 罰則

### 8-1. 即チョンボ

以下が発覚した瞬間: 5枚オール ＋ 逆シュバサイコロチャンス ＋ 聴牌者への放銃扱いチップ支払い

| 主体 | 追加処理 |
|---|---|
| 親の即チョンボ | 本場そのまま流局かつ親流れ |
| 子の即チョンボ | 本場そのままでやり直し |
| オーラス親の即チョンボ | 流局だが親流れなし |

即チョンボになるケース:
- 誤倒牌
- 誤リーチ（発覚次第）
- カンを忘れたままアガリまたは流局
- 白ぽっちをこぼす（ぽっちこぼし、罰則あり＋効果処理あり）

### 8-2. その他罰則

| 事象 | 処理 |
|---|---|
| 誤発声（誤ロン含む） | 1000点供託のみ |
| 誤リーチ取り消し | 下家の打牌完了前なら 1000点で取り消し可 |
| 多牌 | バレ次第ランダムに1牌捨て |
| 少牌 | 申告の上ツモり忘れ箇所からツモ |
| 立直前に他家の手牌を覗く | アガリ放棄 |

## 9. Q&A（公式回答済み）

Q: 配牌でカン8m聴牌の場合、第一ツモぽっちやロンは?
A: 一律アガリ不可。間八萬は立直が聴牌の要件のため。

Q: フィーバーリーチ中、正ぽっちから逆ぽっちをツモった場合、以降の払いは通常アガリ牌でも逆になるか?
A: 以降のアガリは全て逆払い。ただし正ぽっちで逆転可能。

Q: ダブロンで片方が秋、片方が冬の場合は?
A: 秋のドラ表の部分からスタート。金北アガリ者がいる場合は金北処理を先に。

Q: ぽっちこぼし時に誰も立直かかっていない場合は?
A: 正逆どちらも全員に対して効果が消滅する。

Q: 間八萬で緑ロン和了を見逃して青を引いた場合?
A: 立直後の見逃しが認められるので、通常の青ツモとなる。
